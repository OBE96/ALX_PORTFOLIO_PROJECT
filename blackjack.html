The Blackjack Game project is a fully functional web-based implementation of the classic card game Blackjack. This project simulates the game where players compete against a dealer, aiming to get as close to 21 points as possible without exceeding it. The game includes all the typical features, such as hitting, standing, betting, and handling various game scenarios like busts, wins, and ties.

Purpose: This project was developed to demonstrate my understanding of JavaScript logic and DOM manipulation, as well as to showcase my ability to build an interactive game with a clear user interface. It simulates a real-world card game while providing a fun and engaging user experience.
